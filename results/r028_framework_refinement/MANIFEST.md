# R028 Framework Refinement Manifest

Date: 2026-07-02

Purpose: refine the paper framework before drafting. This package does not write
manuscript prose, add new experiments, or merge citation candidates. It fixes
the paper's identity, claim boundaries, section roles, and reviewer-risk map.

| File | Purpose |
|---|---|
| `FRAMEWORK_DECISION.md` | Records the accepted paper framing and alternatives rejected. |
| `CLAIM_BOUNDARY_TABLE.md` | Defines what each claim may say, what evidence supports it, and what not to overclaim. |
| `SECTION_ARCHITECTURE.md` | Assigns each paper section a job, evidence source, and exclusion list. |
| `REVIEWER_OBJECTION_MAP.md` | Anticipates likely reviewer objections and minimum fixes. |
